Shortcuts

  • Scrollwheel = Change the length of a note or rest
  • ctrl+N = Opens new Sheet Wizard
  • ctrl+O = Opens a saved.jssheet file
  • ctrl+S = Saves a .jssheet file
  • ctrl+E = Exports a .jssheet file to MIDI
  • ctrl+P = Prints the current sheet
  • alt+F4 = Exits the software
  • F1 = help
  • +,- = change length of notes and rests
