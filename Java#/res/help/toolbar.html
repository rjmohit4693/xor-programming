Toolbar

ImageFunction Description
Play/Stop Starts or stops the sheet from playing. It is possible to edit while playing.
Add Note Sets your cursor to add notes where you click.
Add Rest Sets your cursor to add rests where you click.
Erase Sets your cursor to replace clicked notes with the same length rest.
Add Sharp Sets your cursor to toggle sharps on clicked notes.
Add Slur Slurs the clicked note with the next one.
Add Measure Adds a measure to each staff.
Remove Measure Removes a measure from each staff.
Add Staff Creates a new staff of whole rests.
Remove Staff Removes a staff.
N/ATempo Slider Allows you to select the tempo for playback in beats per minute.
